# my-daily-notes

## 插件说明

这是一个Obsidian插件，用于管理日记、周记、项目和会议记录。以下是它的主要功能：
- 启用插件时自动生成当天的日记文件
- 昨日未完成的任务会自动添加到当天日记中
- 日记、周记、项目、会议都支持自定义模板
- 提供了一些快捷按钮和命令，方便快速创建日记、周记、项目、会议内容
